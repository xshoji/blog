Macの画面録画を右クリックから簡単にエンコードできるようにするTips

自作のツールの動きを見せたい時や、動作の証跡を残したい時などにMacの画面録画機能で動画を撮ることがあります。ただ、Macの録画機能で撮影された動画はそのままだと結構重いので、SlackやGithubに載せるとなるとエンコードしないとそのままでは扱いづらいです。ターミナルからコマンドを思い出して、ffmpeg等でエンコードすれば良いのですが、できればもっと簡単にサクッとやりたいものです。

そこで今回は、Macで画面を録画した後、その動画ファイルの右クリックメニューからサクッとエンコードしてお手軽にデモ動画を作る方法を紹介します。

準備するもの

  • エンコーダー（ffmpeg, HandBrakeCLIなど）
  • Automatorによるクイックアクション
  • 画面録画した動画ファイル

それぞれ順番に説明していきます。
